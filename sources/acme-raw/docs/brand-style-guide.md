# Acme Brand Style Guide (DISTRACTOR / non-data)

Primary color Acme Indigo #4C36E0; secondary Slate #2B2F3A. Logo clear-space = height of the
"A". Headlines in Söhne, body in Inter. Never stretch the logo or place it on busy photos.
Tone of voice: confident, plain-spoken, never hypey. Email signature template in Notion.
